buddy.auth.protocols

Main authentication and authorization abstractions defined as protocols.

IAuthentication

protocol

Protocol that defines unfied workflow steps for all authentication backends.

members

-authenticate

(-authenticate _ request data)

Given a request and parsed data (from previous step) and try authenticate this data.

If this method returns not nil value, the request will be considered authenticated and the value will be attached to request under :identity attribute.

-parse

(-parse _ request)

Parse token from the request. If it returns nil the authenticate phase will be skipped and the handler will called directly.
