buddy.auth.backends.httpbasic

The http-basic authentication and authorization backend.

http-basic-backend

(http-basic-backend & [{:keys [realm authfn unauthorized-handler], :or {realm "Buddy Auth"}}])

Create an instance of the http-basic based authentication backend.

This backends also implements authorization workflow with some defaults. This means that you can provide own unauthorized-handler hook if the default not satisfies you.
